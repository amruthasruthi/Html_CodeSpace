Frond End Description

The frontend of the application is built using ASP.NET MVC with Razor views for server-side rendering.
It utilizes HTML5, CSS3, and Bootstrap for responsive design, and JavaScript/jQuery for dynamic user
interactions. The UI provides a clean and intuitive experience for managing plans, projects, and tasks.

Back End Description

The backend of the application is developed using ASP.NET Core Web API and follows a Repository and
Controller pattern for clean separation of concerns. The system uses Entity Framework Core (EF Core)
for Object-Relational Mapping (ORM) to interact with the SQL Server database
